mac_youtube_history module¶

Downloads, analyzes, and reports all Youtube videos associated with a user’s Google account.

class mac_youtube_history.Analysis(takeout=None, outpath='data', delay=0)¶

Bases: object

Analysis is responsible for downloading and analyzing the takeout data from google.

Parameters

  • takeout (str) – Path to an unzipped Takeout folder downloaded from https://takeout.google.com/

  • outpath (str, optional) – path to the directory where both raw and computed results should be stored, defaults to data

  • delay (float) – Amount of time in seconds to wait between requests, defaults to 0
